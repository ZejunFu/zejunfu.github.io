Learning Outcomes

  • Understand and use notation for sets and subsets.
  • Recognize and use well-known finite and infinite sets.
  • Understand and use the operrations of set union, intersections, and difference.
  • Represent set operations using Venn diagrams.

Basic Definitions

1.1

A set is a collection of distinct objects that is thought of as a whole. The objects in a set are called elements or members of the set.

1.2

Let $A$ and $B$ be sets. We say that $A$ is a subset of $B$ if every element of $A$ is an element of $B$, and we denote this by writing $A \subset B$

Sometimes, the notation A⊆BA \subseteq BA⊆B is used to denote that AAA is a subset of BBB. (It’s a little bit wrong. :\ ) Then A⊆BA \subseteq BA⊆B explicitly states that AAA is a proper subset of BBB. (There is at least one element of BBB which is not in AAA)
